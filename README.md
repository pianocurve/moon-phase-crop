# Moon Phase Crop

`Moon Phase Crop` は、アップロードした画像に対し、天文学的な計算に基づいた「月相」と「傾き（立ち具合）」をシミュレートし、高品質な透過PNGとして出力するブラウザベースのツールです。

月を単なる図形として扱うのではなく、天球上の位置、観測時期、季節変動に基づいた「自然な月の見え方」をシミュレートすることに重点を置いています。

## 特徴

* **天文学的シミュレーション:** 月相（Phase）だけでなく、観測地（北緯35度基準）と季節、時刻に応じた月の「立ち具合（傾き）」をリアルタイムで計算・反映します。
* **直感的な天球UI:** 太陽と月が二重軌道を回る直感的なシミュレーターを搭載。ドラッグ操作により、天球を回す感覚で時刻と月相を制御できます。
* **柔軟なエクスポート:** 任意の解像度で画像を生成。Glow（光彩）エフェクトの強度調整や、28段階の月相をZIP形式で一括書き出し可能です。
* **高解像度対応:** Webブラウザ上で最大8000pxの解像度まで生成可能で、印刷物などの高精細な用途にも対応します。

## 主な機能

1. **Celestial Sphere Simulator:** 太陽と月の位置関係を視覚化。ドラッグによる時間操作に対応。
2. **Dynamic Tilt Engine:** 月齢と時刻から、その時空における月の傾きを幾何学的に算出。
3. **Bulk Export:** 28フェーズを一括で生成し、ZIP形式でダウンロード。

## 使い方

1. **画像アップロード:** 変換したい画像をドラッグ＆ドロップ（またはタップ）で読み込みます。
2. **設定調整:** Canvasの解像度、Glow強度、観測月、時刻を設定します。
3. **時間操作:** シミュレーター上の天球をドラッグし、希望する時刻・状況に合わせます。
4. **出力:** 「Save Current Phase」で単体保存、または「Save All 28 Default Phases」で全フェーズを一括出力します。

## 開発のヒント

このツールは、以下の技術スタックを組み合わせて構築されています。

* **Canvas API:** 高精細な画像処理とリアルタイム描画。
* **JSZip:** ブラウザ上でのZIPファイル生成処理。
* **球面三角法:** 月の傾き（天頂位置角）の動的計算アルゴリズム。
